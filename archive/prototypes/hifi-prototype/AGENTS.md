# Hi-Fi Prototype Rules

This directory contains an early HTML/JS visual prototype.

## Default behavior

- Preserve it by default.
- Do not refactor, modernize, or reorganize it unless the user explicitly asks for prototype work.

## Rules

- Treat it as a reference artifact, not the production client architecture.
- Do not infer final runtime structure from prototype shortcuts.
- If a change is requested here, keep it explicitly scoped to prototype goals.
