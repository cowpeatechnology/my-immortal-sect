# 01_修仙宗门经营要素与完整度审视

## 1. 结论

基于当前项目方向——**竖屏 2.5D、微信/抖音小游戏、宗门经营为核心、神降为外部输入系统**——第一版本的修仙宗门经营要素，建议分为三层：

### A. 必须在 V1 就立住的核心经营层
1. 弟子
2. 建筑
3. 资源
4. 生产
5. 境界成长
6. 宗门扩张

### B. 应该在 V1 预留接口，但可以做轻的增强层
1. 炼丹
2. 炼器
3. 法宝
4. 灵植/药田
5. 养殖/灵兽
6. 功法/职业倾向

### C. 明显可以后置的扩展层
1. 复杂战斗
2. 大规模 PvP
3. 神降战场
4. 真人共创世界
5. 深度弟子关系网
6. 非核心支线系统（婚配、复杂洞府装饰、长篇剧情分支）

## 2. 你当前提出的要素是否完整

你提出的这些关键词：

- 弟子天赋
- 属性
- 修炼
- 法宝
- 炼丹
- 炼器
- 境界
- 建设
- 养殖

**方向是对的，但还不完整。**

还必须补上 5 个要素，否则很难形成真正的“宗门经营感”：

### 2.1 资源与库存
没有统一的资源、库存、运输、消耗、产出体系，建筑和弟子都会变成空壳。

### 2.2 工作位与岗位分工
必须让弟子能被分配为：
- 外门采集
- 药田照料
- 炼丹
- 炼器
- 讲经/修炼辅助
- 仓储搬运
- 巡山/护宗

### 2.3 建筑升级与加工链
建筑不能只是“摆出来加数值”，而要具备：
- 等级
- 容量
- 速度
- 输入
- 输出
- 前置条件

### 2.4 宗门声望 / 香火 / 扩张资源
这是后续神降回流、玩家长期追求、宗门成长上限的承载位。

### 2.5 弟子标签与随机性
不是只有“数值面板”，还要有：
- 灵根/资质
- 性格/倾向
- 擅长工种
- 命格/体质
- 培养方向

## 3. 近 3 年热门修仙游戏 / 小说，对我们最有借鉴价值的设定

### 3.1 游戏侧
#### 《最强祖师》
- 宗门祖师视角
- 招募弟子
- 弟子随机属性/命格
- 仙田、炼丹、炼器、挖矿等生产岗位
- 模拟经营 + 修仙包装

#### 《一念逍遥》
- 修炼方向（炼体 / 修法）
- 灵植、灵兽、炼丹、炼器、本命法宝
- 洞府/家园式养成
- 境界与材料长期循环

#### 《了不起的修仙模拟器》
- 上帝视角
- 弟子培养
- 功法、灵药、法宝、建筑、修行宝地
- 境界突破与渡劫
- 宗门从聚落成长为大派

#### 《修仙家族模拟器》
- “家族/宗门上位视角”
- 岛屿/据点扩张
- 秘境、探索、资源占领
- 势力成长

#### 《鬼谷八荒》
- 标准修仙境界线
- 功法流派
- 奇遇、妖兽、法宝、抉择
- 山海经式世界质感

### 3.2 小说侧
#### 《凡人修仙传》
- 资质不高但靠积累成长
- 炼丹、灵药、资源经营感
- 境界与材料高度绑定
- 修仙并非无脑升级，而是资源管理

#### 《光阴之外》
- 宗门/联盟结构
- 生存压力
- 世界层级与势力秩序
- 更强烈的残酷成长氛围

#### 《谁让他修仙的！》
- 宗门、师门、灵根、特殊体质等经典标签
- 轻松、反差、角色个性记忆点
- 弟子/角色容易出“梗”

#### 《苟在妖武乱世修仙》
- 灵农、坊市、底层资源循环
- 灵植、炼丹、阵法、修仙副职业
- 生存与经营并重

#### 《玄鉴仙族》《我的模拟长生路》
- 家族/势力经营感
- 天赋、词条、传承
- 长线成长与代际结构

## 4. 第一版本应锁定的“完整经营要素清单”

### 4.1 V1 必做要素
#### 弟子
- 灵根
- 资质
- 属性
- 擅长岗位
- 境界
- 当前工作
- 装备位（先少量）
- 忠诚/心情（可先简化）

#### 建筑
- 宗门大殿
- 弟子居
- 仓库
- 药田/灵田
- 炼丹房
- 炼器房
- 采集建筑
- 练功场 / 讲经阁

#### 资源
- 木材
- 石料
- 粮食
- 灵草
- 矿石
- 丹药原料
- 炼器原料
- 灵气/香火

#### 生产链
- 采集
- 入库
- 加工
- 消耗
- 产出
- 升级材料消耗

#### 成长
- 修为获取
- 境界突破
- 装备/法宝加成（先轻）
- 建筑升级
- 宗门地块扩张

#### 神降（V1 定位）
- 作为外部事件入口
- 带回资源 / 香火 / 特殊材料 / 弟子机缘
- 不作为第二主玩法

## 5. 最终判断

你的原始清单已经抓住了“修仙味”，但若想做成**真正能长期扩展的宗门经营**，必须把这些重新组织成一句话：

**弟子驱动生产，建筑承载岗位，资源支撑修炼，境界反馈成长，神降负责外部输入。**
