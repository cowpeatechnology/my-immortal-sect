# Workspace

This directory holds local outputs, validation artifacts, and temporary working files.

Current usage:

- `output/browser/`: browser automation captures, ChatGPT bridge results, HAR files, and stress-test outputs
- `output/cocos/`: local Cocos-related generated artifacts that are not part of the tracked product source
- `output/assets/`: generated source images and similar asset-side working outputs
- `runtime/gameplay/`: local gameplay validation screenshots and similar runtime evidence

Keep product source code, long-lived docs, and reusable tools outside this directory.
